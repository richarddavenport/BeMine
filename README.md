# Be Mine - Valentine's Day Website

A sweet and girly Valentine's Day website for that special someone! 

## Features
- Beautiful pink gradient background with floating hearts
- Two interactive pages
- Smooth animations and transitions
- Fun "No" button that tries to escape
- Confetti celebration on "Yes"
- Fully responsive design

## Pages
1. **Page 1**: The question "Will you be my Valentine?" with Yes/No buttons
2. **Page 2**: Celebration page with "Hooray! Best decision ever!"

## How to Use
Simply open `index.html` in your browser to get started!

## Customization
- Edit the name "Lottie" in `index.html` to personalize it
- Modify colors in `styles.css` to change the theme
- Update messages in `index.html` as desired
